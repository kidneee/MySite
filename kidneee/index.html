全てのユーザーに優しいサイトを目指して。

Webは、どんな障害を持っていようが、自分で情報を得ることができる唯一のメディア。 そんなWebが私は大好きです。
そして、見る人に優しいサイトをたくさん作って、色んな人に適切な情報をお届けしたい。それが私にできる『福祉』だと思っています。
HTMLとCSSにとことんこだわって、アクセシビリティの良いサイトを作ることが目標です。

Webサイト制作に欠かせないスキル

HTML

Webサイトの土台、HTML。正しくマークアップしていくことで、誰もが情報を得やすいサイトになります。「HTMLに始まり、HTMLに終わる」、そう肝に銘じております。

CSS

Webサイトの見た目（デザイン）を作る言語・CSS。シンプルなサイトからおしゃれ度100%なサイトまで、この言語を使いこなせない事には意味がありません。
